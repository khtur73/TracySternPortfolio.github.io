Description:

Use scratch to design and create a program that shows my name. The requirements were to add a sprite for each letter in my name and give them 3 unique behaviors. The program needed to run when the green flag was clicked and for extra credit I added that the letter C will represent John Cena and his name is on the screen, and there’s his music and then is head move on the screen.

Concepts Learned:

  • Using event blocks - green flag clicked, when key pressed
  • Conditional statements - used if statements to make them move one after another
  • Used loops - I used a forever block to make one of them to change colors forever, I used a repeat until block to make one of them repeat something couple times.
  • Sprites - I created my own sprites using the sprite editor, I used the library of sprites to find letters in my name and adjusted the costumes, and create 5 letters.
  • Movement - I used the x and y values to make my sprites move to their original location when the green flag was clicked, I rotated the sprites and moved them around.
